# Localization (i18n)

Translations are stored in `i18n/*.json` and loaded by `locale` in `config.json`.

Available locales in this repo:

- `en`
- `it`
- `es`
- `fr`
- `de`
- `pt`

## Scope

Translation files are for fixed UI strings only:

- labels (`OUTDOOR`, `ROOMS`, `TEMP`, `HUM`, ...)
- weather condition names
- weekday/month names
- intraday phase labels
- fallback quote text

User customizations stay in `config.json`:

- `header_title`
- entities and room names
- refresh behavior settings

## Structure

Each locale file can override these keys:

- `weekdays_abbr` (7 items)
- `weekdays_full` (7 items)
- `months_abbr` (12 items)
- `months_full` (12 items)
- `intraday_labels` (3 items)
- `condition_labels` (map)
- `labels` (map)
- `fallback_quote` (`text`, `author`)

Invalid sections automatically fall back to English defaults.
